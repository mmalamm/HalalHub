# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Trucks

- `GET /api/trucks`
  - Trucks index/search
- `POST /api/trucks`
- `GET /api/trucks/:id`
- `PATCH /api/trucks/:id`
- `DELETE /api/trucks/:id`

### Reviews

- `GET /api/reviews`
- `POST /api/reviews`
- `GET /api/reviews/:id`
- `DELETE /api/reviews/:id`
- `GET /api/trucks/:id/reviews`
